  • First you are going to want to install python onto your machine if you haven't already
  • On google search install python, click the first link by www.python.org
  • Select Windows, where it says "Looking for python with a different OS? Python for Windows."
  • On the installer, hit install now, and don't forget to check the box, *add python 3.8 to path* this alleviates you having to manually add the path.
  • Once finished installing, open your command line, and type in the command line: Pip install Django == 2.2.7
  • Now, once the install is finished, type python into your command line, you should see three >>> keys
  • Type import django, press enter
  • Type print(django.get_version())
  • After doing this, you should be able to see the versino of django that you just installed
